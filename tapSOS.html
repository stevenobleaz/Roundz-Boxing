TapSOS Emergency Notification System - Consent Documentation 1. Purpose: - TapSOS is an emergency notification system that allows users to send alerts to their emergency contacts - Messages are only sent when the user explicitly triggers the SOS button - All messages are opt-in, as users must manually add their emergency contacts 2. Consent Process: - Users must explicitly add each emergency contact through the app's interface - Contacts are only added after user confirmation - Users can remove contacts at any time - Each contact must be manually added and verified by the user 3. Message Content: - Emergency alerts only - Location information (if enabled) - Timestamp of the emergency - No marketing or promotional content 4. Opt-out Process: - Users can remove contacts at any time through the app - Contacts can request removal by contacting the user
